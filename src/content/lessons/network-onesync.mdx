## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_network_onesync
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_network_onesync/
  fxmanifest.lua
  server.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

server_script 'server.lua'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
RegisterCommand('bucket', function(src, args)
    local bucket = tonumber(args[1]) or 0
    SetPlayerRoutingBucket(src, bucket)
    print(('[qu_network_onesync] moved %s to bucket %d'):format(GetPlayerName(src), bucket))
end, false)

RegisterCommand('mybucket', function(src)
    print('[qu_network_onesync] bucket ' .. GetPlayerRoutingBucket(src))
end, false)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_network_onesync
    ```

    Save, then run:

    ```text
    restart qu_network_onesync
    ```

    Run this test:

    ```text
    /bucket 10 then /mybucket
    ```

    <ExpectedOutput>
      [qu_network_onesync] moved yourName to bucket 10
[qu_network_onesync] bucket 10
    </ExpectedOutput>
  </Step>
</StepGroup>
