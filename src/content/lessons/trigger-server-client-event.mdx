## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_trigger_server_client_event
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_trigger_server_client_event/
  fxmanifest.lua
  server.lua
  client.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

client_script 'client.lua'
server_script 'server.lua'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
RegisterNetEvent('qu_trigger_server_client_event:ping', function(message)
    local src = source
    if type(message) ~= 'string' or #message > 40 then return end
    print('[qu_trigger_server_client_event] server got ' .. message .. ' from ' .. GetPlayerName(src))
    TriggerClientEvent('qu_trigger_server_client_event:pong', src, 'server received ' .. message)
end)
    ```

    Open `client.lua` and paste this:

    ```lua
RegisterCommand('pingserver', function(_, args) TriggerServerEvent('qu_trigger_server_client_event:ping', table.concat(args, ' ')) end, false)
RegisterNetEvent('qu_trigger_server_client_event:pong', function(reply) print('[qu_trigger_server_client_event] ' .. reply) end)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_trigger_server_client_event
    ```

    Save, then run:

    ```text
    restart qu_trigger_server_client_event
    ```

    Run this test:

    ```text
    /pingserver hello
    ```

    <ExpectedOutput>
      Server: [qu_trigger_server_client_event] server got hello from yourName
F8: [qu_trigger_server_client_event] server received hello
    </ExpectedOutput>
  </Step>
</StepGroup>
