## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_vehicles_spawn_modify_store
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_vehicles_spawn_modify_store/
  fxmanifest.lua
  client.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

client_script 'client.lua'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `client.lua` and paste this:

    ```lua
local testVehicle
RegisterCommand('testvehicle', function()
    if testVehicle and DoesEntityExist(testVehicle) then DeleteEntity(testVehicle) end
    local model = joaat('sultan')
    RequestModel(model) while not HasModelLoaded(model) do Wait(0) end
    local ped = PlayerPedId()
    local c = GetEntityCoords(ped)
    testVehicle = CreateVehicle(model, c.x + 3.0, c.y, c.z, GetEntityHeading(ped), true, false)
    SetVehicleNumberPlateText(testVehicle, 'QUCAR')
    SetVehicleColours(testVehicle, 12, 12)
    TaskWarpPedIntoVehicle(ped, testVehicle, -1)
    SetModelAsNoLongerNeeded(model)
    print('[qu_vehicles_spawn_modify_store] spawned sultan')
    SetTimeout(60000, function() if testVehicle and DoesEntityExist(testVehicle) then DeleteEntity(testVehicle) print('[qu_vehicles_spawn_modify_store] auto-deleted') end end)
end, false)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_vehicles_spawn_modify_store
    ```

    Save, then run:

    ```text
    restart qu_vehicles_spawn_modify_store
    ```

    Run this test:

    ```text
    /testvehicle
    ```

    <ExpectedOutput>
      F8: [qu_vehicles_spawn_modify_store] spawned sultan
F8 after 60 seconds: [qu_vehicles_spawn_modify_store] auto-deleted
    </ExpectedOutput>
  </Step>
</StepGroup>
