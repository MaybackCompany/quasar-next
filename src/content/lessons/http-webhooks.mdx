## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_http_webhooks
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_http_webhooks/
  fxmanifest.lua
  server.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

server_script 'server.lua'
    ```

    Add these lines to server.cfg above `ensure qu_http_webhooks`:

    ```text
set qu_http_webhooks_webhook YOUR_DISCORD_WEBHOOK_URL
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
local webhook = GetConvar('qu_http_webhooks_webhook', '')

RegisterCommand('webhooktest', function(src)
    if webhook == '' then print('[qu_http_webhooks] missing webhook') return end
    local name = src == 0 and 'console' or GetPlayerName(src)
    PerformHttpRequest(webhook, function(code)
        print('[qu_http_webhooks] Discord status ' .. code)
    end, 'POST', json.encode({ username = 'qu_http_webhooks', content = 'test from ' .. name }), { ['Content-Type'] = 'application/json' })
end, false)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_http_webhooks
    ```

    Save, then run:

    ```text
    restart qu_http_webhooks
    ```

    Run this test:

    ```text
    /webhooktest
    ```

    <ExpectedOutput>
      [qu_http_webhooks] Discord status 204
    </ExpectedOutput>
  </Step>
</StepGroup>
