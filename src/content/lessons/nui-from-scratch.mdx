>
  **NUI** is FiveM's name for the in-game browser layer, a real Chromium tab the client renders on top of GTA. Every menu, hud, scoreboard, F-pause UI you have ever seen in a server is NUI. So far you have used `ox_lib` and other framework UIs that wrap NUI for you. This bonus shows the layer underneath: how to declare a page in `fxmanifest.lua`, how to push state from Lua with `SendNUIMessage`, how to receive clicks with `RegisterNUICallback`, and how to capture the cursor with `SetNuiFocus`. Once you understand these four pieces, you can read any NUI resource and build your own without a framework.

## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_nui_from_scratch
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_nui_from_scratch/
  fxmanifest.lua
  client.lua
  web/index.html
  web/app.js
  web/style.css
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

client_script 'client.lua'
ui_page 'web/index.html'
files {
    'web/index.html',
    'web/app.js',
    'web/style.css'
}
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `client.lua` and paste this:

    ```lua
RegisterCommand('openui', function()
    SetNuiFocus(true, true)
    SendNUIMessage({ show = true })
    print('[qu_nui_from_scratch] opened')
end, false)

RegisterNUICallback('close', function(_, cb)
    SetNuiFocus(false, false)
    print('[qu_nui_from_scratch] closed')
    cb({ ok = true })
end)
    ```

    Open `web/index.html` and paste this:

    ```text
<!doctype html>
<html><body><button id="close">Close</button><script src="app.js"></script></body></html>
    ```

    Open `web/app.js` and paste this:

    ```text
document.getElementById('close').addEventListener('click', () => { fetch('https://qu_nui_from_scratch/close', { method: 'POST', body: '{}' }) })
    ```

    Open `web/style.css` and paste this:

    ```text
body { display: grid; place-items: center; min-height: 100vh; }
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_nui_from_scratch
    ```

    Save, then run:

    ```text
    restart qu_nui_from_scratch
    ```

    Run this test:

    ```text
    /openui, then click Close
    ```

    <ExpectedOutput>
      F8: [qu_nui_from_scratch] opened
F8: [qu_nui_from_scratch] closed
    </ExpectedOutput>
  </Step>
</StepGroup>
