>
  A **convar** (configuration variable) is a server-side key/value pair you set in `server.cfg` or at runtime, then read from any resource. Convars are how you externalize things that *change per server*, debug flags, database name, max stack size, Discord invite, owner ID, without editing Lua and without rebuilding the resource. Three flavors decide who sees them, the API is four functions, and you already use them every time you write `sv_hostname`.
</LessonHero>

## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_convars_server_cfg
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_convars_server_cfg/
  fxmanifest.lua
  server.lua
  client.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

client_script 'client.lua'
server_script 'server.lua'
    ```

    Add these lines to server.cfg above `ensure qu_convars_server_cfg`:

    ```text
set qu_convars_server_cfg_cash 750
setr qu_convars_server_cfg_debug true
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
RegisterCommand('cashcfg', function()
    print('[qu_convars_server_cfg] cash ' .. GetConvarInt('qu_convars_server_cfg_cash', 500))
end, true)
    ```

    Open `client.lua` and paste this:

    ```lua
RegisterCommand('debugcfg', function()
    print('[qu_convars_server_cfg] debug ' .. GetConvar('qu_convars_server_cfg_debug', 'false'))
end, false)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_convars_server_cfg
    ```

    Save, then run:

    ```text
    restart qu_convars_server_cfg
    ```

    Run this test:

    ```text
    cashcfg, then /debugcfg
    ```

    <ExpectedOutput>
      [qu_convars_server_cfg] cash 750
F8: [qu_convars_server_cfg] debug true
    </ExpectedOutput>
  </Step>
</StepGroup>
