## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_anticheat
    ```

    Keep the spelling exact. You will use the same name in `server.cfg` and in the restart command.
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_anticheat/
  fxmanifest.lua
  server.lua
    ```

    If a subfolder does not exist yet, create it before you create the file inside it.
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this exact manifest:

    ```lua
fx_version 'cerulean'
game 'gta5'

server_script 'server.lua'
    ```

    Add these lines near your other config values:

    ```text
set qu_ac_webhook YOUR_DISCORD_WEBHOOK_URL
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The resource contains working FiveM code for this topic.">
    Open `server.lua` and paste this:

    ```lua
local lastUse = {}
local webhook = GetConvar('qu_ac_webhook', '')

local function logViolation(src, reason)
    local name = GetPlayerName(src) or 'unknown'
    print(('[qu_anticheat] blocked %s: %s'):format(name, reason))

    if webhook ~= '' then
        PerformHttpRequest(webhook, function() end, 'POST', json.encode({
            username = 'qu_anticheat',
            content = ('Blocked %s: %s'):format(name, reason)
        }), { ['Content-Type'] = 'application/json' })
    end
end

RegisterNetEvent('qu_anticheat:claimReward', function(amount)
    local src = source
    local now = os.time()

    if lastUse[src] and now - lastUse[src] < 10 then
        logViolation(src, 'reward spam')
        return
    end

    if type(amount) ~= 'number' or amount < 1 or amount > 500 then
        logViolation(src, 'bad amount ' .. tostring(amount))
        return
    end

    lastUse[src] = now
    print(('[qu_anticheat] reward approved for %s: %d'):format(GetPlayerName(src), amount))
end)

RegisterCommand('actest', function(src)
    if src == 0 then
        print('[qu_anticheat] use /actest in game')
        return
    end

    TriggerEvent('qu_anticheat:claimReward', 250)
end, false)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="You see the proof line in the correct console.">
    Open `server.cfg` and add this line near your other local resources:

    ```text
    ensure qu_anticheat
    ```

    Save the file. In txAdmin Live Console or the server console, run:

    ```text
    restart qu_anticheat
    ```

    Now run this test:

    ```text
    /actest
    ```

    <ExpectedOutput>
      [qu_anticheat] reward approved for yourName: 250
    </ExpectedOutput>
  </Step>
</StepGroup>
