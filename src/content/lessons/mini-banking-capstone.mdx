## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_mini_banking_capstone
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_mini_banking_capstone/
  fxmanifest.lua
  server.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

server_script '@oxmysql/lib/MySQL.lua'
server_script 'server.lua'

dependency 'oxmysql'
    ```

    Run this SQL before starting the resource:

    ```sql
CREATE TABLE IF NOT EXISTS qu_mini_banking_capstone_rows (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(80) NOT NULL,
  amount INT NOT NULL DEFAULT 0
);
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
RegisterCommand('dbtest', function(src)
    local id = MySQL.insert.await('INSERT INTO qu_mini_banking_capstone_rows (name, amount) VALUES (?, ?)', { 'lesson', 100 })
    local row = MySQL.single.await('SELECT * FROM qu_mini_banking_capstone_rows WHERE id = ?', { id })
    print('[qu_mini_banking_capstone] selected ' .. row.name .. ' ' .. row.amount)
    MySQL.update.await('UPDATE qu_mini_banking_capstone_rows SET amount = ? WHERE id = ?', { 150, id })
    print('[qu_mini_banking_capstone] updated ' .. id)
end, true)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_mini_banking_capstone
    ```

    Save, then run:

    ```text
    restart qu_mini_banking_capstone
    ```

    Run this test:

    ```text
    dbtest
    ```

    <ExpectedOutput>
      [qu_mini_banking_capstone] selected lesson 100
[qu_mini_banking_capstone] updated rowId
    </ExpectedOutput>
  </Step>
</StepGroup>
