## Build it

<StepGroup>
  <Step num={1} title="Download FiveM server artifacts" outcome="FXServer.exe is on your computer.">
    Go to https://runtime.fivem.net/artifacts/fivem/build_server_windows/master/

    Download the **Recommended** build (green label, NOT the top "Latest"). Extract the zip to:

    ```text
C:\FXServer\
    ```

    Expected contents: `FXServer.exe`, `citizen/` folder, `.dll` files.

    <ExpectedOutput>
      Double-clicking `FXServer.exe` opens a console window, then txAdmin starts.
    </ExpectedOutput>
  </Step>

  <Step num={2} title="Get your license key" outcome="You have the one mandatory key.">
    Go to https://portal.cfx.re/ → **Servers** → **Create Server**. Enter a name. Copy the generated key.

    The key looks like: `licensekey:cfxk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

    This is required even for local/single-player servers. Without it, the server won't start.
  </Step>

  <Step num={3} title="Write a minimal server.cfg" outcome="The server has a name, endpoints, and a license.">
    Create `server.cfg` in `C:\FXServer\`:

    ```bash server.cfg
# How players reach your server
endpoint_add_tcp "0.0.0.0:30120"
endpoint_add_udp "0.0.0.0:30120"

# Your license key from portal.cfx.re
sv_licenseKey "your_license_key_here"

# What players see in the browser list
sv_hostname "My First FiveM Server"

# Enable OneSync (required for modern servers)
onesync on

# Maximum players
sv_maxclients 32
    ```

    Six lines. That's it. You don't need tags, Discord links, or resource lists yet.
  </Step>

  <Step num={4} title="Start the server" outcome="The server console shows it's running.">
    Double-click `FXServer.exe`. The console window opens. You'll see:

    - "Validating license key" → "License key OK"
    - txAdmin starting at `localhost:40120`
    - "Server ready" message after ~30 seconds

    Don't worry about txAdmin right now. Just watch the console until it says the server is ready.

    <ExpectedOutput>
      Console prints: `[txAdmin] Server is ready.`
    </ExpectedOutput>
  </Step>

  <Step num={5} title="Join your server" outcome="You're standing in Los Santos on your own server.">
    Open FiveM. Press F8 to open console. Type:

    ```text
connect 127.0.0.1:30120
    ```

    Or if you want friends on your local network to join:

    ```text
connect 192.168.x.x
    ```

    (Replace with your PC's local IP — find it with `ipconfig` in Command Prompt.)

    <ExpectedOutput>
      You load into Los Santos. The server name appears in the bottom-right corner. You did it.
    </ExpectedOutput>
  </Step>
</StepGroup>

## Common failures

| Exact error | Cause | Fix |
|---|---|---|
| `FXServer.exe` window opens then closes instantly | Missing VC++ Redistributable | Download and install VC_redist.x64.exe from Microsoft |
| `Invalid license key` | Key copied wrong or not activated | Re-copy from portal.cfx.re, make sure there are no extra spaces |
| `connect localhost` does nothing | FiveM client not running or server not ready | Wait for "Server is ready" in the console, then try again |
| `Failed to start resource 'txAdmin'` | Port 40120 in use | Close other programs using that port, or restart your PC |
