## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_trust_and_authority
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_trust_and_authority/
  fxmanifest.lua
  server.lua
  client.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

client_script 'client.lua'
server_script 'server.lua'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
RegisterNetEvent('qu_trust_and_authority:claim', function(clientAmount)
    print('[qu_trust_and_authority] ignored client amount ' .. tostring(clientAmount))
    TriggerClientEvent('qu_trust_and_authority:paid', source, 100)
end)
    ```

    Open `client.lua` and paste this:

    ```lua
RegisterCommand('claimreward', function() TriggerServerEvent('qu_trust_and_authority:claim', 999999) end, false)
RegisterNetEvent('qu_trust_and_authority:paid', function(amount) print('[qu_trust_and_authority] approved ' .. amount) end)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_trust_and_authority
    ```

    Save, then run:

    ```text
    restart qu_trust_and_authority
    ```

    Run this test:

    ```text
    /claimreward
    ```

    <ExpectedOutput>
      Server: [qu_trust_and_authority] ignored client amount 999999
F8: [qu_trust_and_authority] approved 100
    </ExpectedOutput>
  </Step>
</StepGroup>
