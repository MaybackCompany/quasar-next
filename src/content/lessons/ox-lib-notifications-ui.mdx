## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_ox_lib_notifications_ui
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_ox_lib_notifications_ui/
  fxmanifest.lua
  client.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

shared_script '@ox_lib/init.lua'
client_script 'client.lua'

dependency 'ox_lib'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `client.lua` and paste this:

    ```lua
RegisterCommand('oxdemo', function()
    lib.notify({ title = 'qu_ox_lib_notifications_ui', description = 'Notification works', type = 'success' })
    lib.registerContext({ id = 'qu_ox_lib_notifications_ui_menu', title = 'qu_ox_lib_notifications_ui', options = { { title = 'Ask name', event = 'qu_ox_lib_notifications_ui:ask' } } })
    lib.showContext('qu_ox_lib_notifications_ui_menu')
end, false)

RegisterNetEvent('qu_ox_lib_notifications_ui:ask', function()
    local input = lib.inputDialog('Name', { 'Name' })
    if not input then return end
    lib.progressCircle({ duration = 1000, label = 'Saving' })
    print('[qu_ox_lib_notifications_ui] saved ' .. input[1])
end)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_ox_lib_notifications_ui
    ```

    Save, then run:

    ```text
    restart qu_ox_lib_notifications_ui
    ```

    Run this test:

    ```text
    /oxdemo
    ```

    <ExpectedOutput>
      F8: [qu_ox_lib_notifications_ui] saved yourName
    </ExpectedOutput>
  </Step>
</StepGroup>
