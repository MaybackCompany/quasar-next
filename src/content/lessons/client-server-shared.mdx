## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_client_server_shared
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_client_server_shared/
  fxmanifest.lua
  config.lua
  server.lua
  client.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

client_script 'client.lua'
server_script 'server.lua'
shared_script 'config.lua'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `config.lua` and paste this:

    ```lua
Config = {}
Config.Label = 'qu_client_server_shared shared config'
    ```

    Open `server.lua` and paste this:

    ```lua
print('[qu_client_server_shared] server loaded ' .. Config.Label)

RegisterCommand('splitproof', function(src)
    TriggerClientEvent('qu_client_server_shared:clientProof', src)
    print('[qu_client_server_shared] server handled splitproof')
end, false)
    ```

    Open `client.lua` and paste this:

    ```lua
print('[qu_client_server_shared] client loaded ' .. Config.Label)

RegisterNetEvent('qu_client_server_shared:clientProof', function()
    print('[qu_client_server_shared] client ped ' .. PlayerPedId())
end)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_client_server_shared
    ```

    Save, then run:

    ```text
    restart qu_client_server_shared
    ```

    Run this test:

    ```text
    /splitproof
    ```

    <ExpectedOutput>
      Server: [qu_client_server_shared] server handled splitproof
F8: [qu_client_server_shared] client ped playerPedId
    </ExpectedOutput>
  </Step>
</StepGroup>
