## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_teleport_menu
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_teleport_menu/
  fxmanifest.lua
  server.lua
  client.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

shared_script '@ox_lib/init.lua'
client_script 'client.lua'
server_script 'server.lua'

dependency 'ox_lib'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
RegisterNetEvent('qu_teleport_menu:request', function(destination)
    if destination ~= 'legion' and destination ~= 'airport' then return end
    TriggerClientEvent('qu_teleport_menu:approved', source, destination)
end)
    ```

    Open `client.lua` and paste this:

    ```lua
local spots = { legion = vec3(215.76, -810.12, 30.73), airport = vec3(-1037.62, -2737.86, 20.17) }
RegisterCommand('tpmenu', function()
    lib.registerContext({ id = 'qu_teleport_menu_menu', title = 'Teleport', options = { { title = 'Legion', onSelect = function() TriggerServerEvent('qu_teleport_menu:request', 'legion') end }, { title = 'Airport', onSelect = function() TriggerServerEvent('qu_teleport_menu:request', 'airport') end } } })
    lib.showContext('qu_teleport_menu_menu')
end, false)
RegisterNetEvent('qu_teleport_menu:approved', function(destination)
    local c = spots[destination]
    DoScreenFadeOut(500) Wait(600) SetEntityCoords(PlayerPedId(), c.x, c.y, c.z, false, false, false, false) DoScreenFadeIn(500)
    print('[qu_teleport_menu] moved to ' .. destination)
end)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_teleport_menu
    ```

    Save, then run:

    ```text
    restart qu_teleport_menu
    ```

    Run this test:

    ```text
    /tpmenu and choose Legion
    ```

    <ExpectedOutput>
      F8: [qu_teleport_menu] moved to legion
    </ExpectedOutput>
  </Step>
</StepGroup>
