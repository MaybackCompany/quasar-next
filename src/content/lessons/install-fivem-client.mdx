<Callout variant="fivem" title="What FiveM is, in one minute">
 GTA V ships with one online mode that Rockstar controls. FiveM is a **separate launcher** (`FiveM.exe`) that uses your legal copy of GTA V to load the game world, then swaps Rockstar's multiplayer code for its own so you can join community servers. Your GTA V copy never changes, and you can still play GTA Online normally any other time.
</Callout>

## Build it

<StepGroup>
 <Step num={1} title="Download the FiveM client" outcome="FiveM.exe is sitting in your Downloads folder.">
 Only download FiveM from the official site. Do not run installers from mirrors, Discord attachments, or video descriptions.

 Open this URL in your browser:

 ```text
 https://fivem.net/
 ```

 Click the **Download Client** button, read FiveM's **Terms of Service**, then click **Sure, I agree!**. The download starts and you end up with one file:

 ```text
 Downloads/FiveM.exe
 ```

 <ExpectedOutput>
 FiveM.exe appears in your Downloads folder (a few MB, not the full game).
 </ExpectedOutput>
 </Step>

 <Step num={2} title="Run the installer and point it at GTA5.exe" outcome="FiveM knows where your GTA V install lives.">
 Double-click `FiveM.exe`. If Windows shows an unexpected SmartScreen or antivirus warning, stop and confirm that the file came directly from `fivem.net`. Do not bypass a warning for a file from any other source; use the Cfx.re support guidance linked below if the official installer is blocked.

 When FiveM asks for your game, browse to your GTA V folder and select **`GTA5.exe`**. The default folders are:

 ```text
 Steam: C:\Program Files (x86)\Steam\steamapps\common\Grand Theft Auto V\
 Epic: C:\Program Files\Epic Games\GTAV\
 Rockstar: C:\Program Files\Rockstar Games\Grand Theft Auto V\
 ```

 Pick `GTA5.exe`, not `PlayGTAV.exe`. They sit side by side in the same folder and are easy to mix up. `PlayGTAV.exe` is the launcher shortcut; `GTA5.exe` is the real game binary FiveM needs. Click **Open** to confirm.

 <ExpectedOutput>
 FiveM accepts the path and starts preparing to update game data. No "could not find game files" error.
 </ExpectedOutput>
 </Step>

 <Step num={3} title="Let FiveM update game data" outcome="FiveM has built its local cache of GTA V assets.">
 A message says *local game data needs to be updated*. Click **Yes**. FiveM caches a stripped, signed copy of the GTA V data so it can boot into a server in seconds instead of re-reading every `.rpf` archive. This is a one-time job per machine.

 Expect a few minutes on an SSD, longer on a hard drive. The progress bar moves visibly. Everything FiveM writes lives here:

 ```text
 %LocalAppData%\FiveM\
 ```

 <ExpectedOutput>
 The FiveM main window opens with a server browser. It can look empty, that is fine.
 </ExpectedOutput>
 </Step>

 <Step num={4} title="Sign in with Cfx.re" outcome="You see a Welcome screen with your username inside FiveM.">
 FiveM shows its **Terms of Service**. Read it and click **I Accept**. A window appears: *Sign in with Cfx.re*. Do not skip this. Cfx.re is FiveM's identity platform, one account covers the forum, your launcher sign-in, and the **Cfx.re Portal** where you will later generate the free license key your own server needs.

 Click **Sign in with Cfx.re**. Your browser opens to:

 ```text
 https://forum.cfx.re/
 ```

 If you do not have an account yet, that is fine, you create one in Step 02 and come back. If you already have one, log in, then click **Authorize**. Authorizing tells Cfx.re "this copy of FiveM on this machine speaks for me". The token only proves your identity to servers you join. It does not hand FiveM your password, email, or billing, and you can revoke it any time at `forum.cfx.re/my/preferences/account`.

 <ExpectedOutput>
 FiveM greets you by name (something like "Welcome, &lt;your username&gt;") and your username shows in the top-right of the main menu. The exact wording may differ between client versions.
 </ExpectedOutput>
 </Step>

 <Step num={5} title="Prove the install with a quick connect" outcome="You can open the F8 console and reach a server.">
 Click **Continue** to land on the FiveM main menu with your username in the top right. The fastest proof is to connect to any public server. Click **Play**, search for one (for example "Grand Theft Hamlet"), and click **Connect**.

 To talk to FiveM directly you use the in-game **F8 console**. Press F8 at any time to open it. This is the same console you will use to connect to your own local server in a later step, once you have one running. Here `127.0.0.1` means "this same computer" (also called localhost) and `30120` is the default port a FiveM server listens on:

 ```text
 connect 127.0.0.1:30120
 ```

 Do NOT run this yet. You have no server of your own running at this point, so it will fail to connect. It is shown here only so you recognise it later.

 For now you only need to confirm the client works. To leave a server, open F8 and run:

 ```text
 disconnect
 ```

 <ExpectedOutput>
 You see the FiveM loading screen, then either spawn in or get a clear message like "you are not whitelisted". Either result proves your install and sign-in work.
 </ExpectedOutput>
 </Step>
</StepGroup>

## Common failures

| What you see | What it means | How to fix it |
|---|---|---|
| FiveM says "Could not find game files" | You pointed it at the wrong folder or file | Browse again and pick `GTA5.exe`, NOT `PlayGTAV.exe`. They sit in the same folder |
| "Game cache needs to update" every time you launch | FiveM didn't finish caching | Let the update bar reach 100%. On a hard drive this can take 15+ minutes. Don't close it |
| SmartScreen or antivirus blocks FiveM.exe | Windows doesn't recognize the file | If you downloaded from fivem.net, click "More info" → "Run anyway". If from anywhere else, delete it |
| FiveM window is black/empty after opening | Server list hasn't loaded yet, or firewall blocked it | Wait 30 seconds. If still empty, check Windows Firewall isn't blocking FiveM |
| "You must sign in with Cfx.re" keeps appearing | You haven't linked your Cfx.re account | Click the notification, sign in at forum.cfx.re, and authorize FiveM |
