## Build it

<StepGroup>
  <Step num={1} title="Make the resource folder" outcome="The server has one folder for this lesson.">
    Inside your server's `resources` folder, create this folder:

    ```text
    resources/qu_functions
    ```
  </Step>

  <Step num={2} title="Create the files" outcome="Every file named in the manifest exists.">
    Create this exact file layout:

    ```text
resources/qu_functions/
  fxmanifest.lua
  server.lua
    ```
  </Step>

  <Step num={3} title="Write fxmanifest.lua" outcome="FiveM knows which files to load.">
    Open `fxmanifest.lua` and paste this:

    ```lua
fx_version 'cerulean'
game 'gta5'

server_script 'server.lua'
    ```
  </Step>

  <Step num={4} title="Write the lesson code" outcome="The topic is now represented by runnable code.">
    Open `server.lua` and paste this:

    ```lua
local function greet(name) return 'hello ' .. name end
local function classify(level) if level >= 10 then return 'admin' end return 'player' end
local function divide(a, b) if b == 0 then return nil, 'division by zero' end return a / b end

RegisterCommand('functiontest', function()
    local value, err = divide(10, 2)
    print('[qu_functions] ' .. greet('Lua'))
    print('[qu_functions] rank ' .. classify(12))
    print('[qu_functions] divide ' .. tostring(value) .. ' err ' .. tostring(err))
end, true)
    ```
  </Step>

  <Step num={5} title="Start and test it" outcome="The expected proof appears in the correct console.">
    Open `server.cfg` and add this line:

    ```text
    ensure qu_functions
    ```

    Save, then run:

    ```text
    restart qu_functions
    ```

    Run this test:

    ```text
    functiontest
    ```

    <ExpectedOutput>
      [qu_functions] hello Lua
[qu_functions] rank admin
[qu_functions] divide 5 err nil
    </ExpectedOutput>
  </Step>
</StepGroup>
