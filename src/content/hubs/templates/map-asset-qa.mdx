## Copyable QA note

```
# Map QA
Resource:
Location:
Files checked: fxmanifest.lua, stream/, ymap, ytyp, ybn

[ ] Starts from clean restart
[ ] No txAdmin errors
[ ] No F8 errors
[ ] Entrance works
[ ] Interior works
[ ] Collision works
[ ] Doors and stairs work
[ ] No duplicate objects
[ ] No nearby ymap conflict
[ ] resmon checked in the area
[ ] Rollback commit ready

Decision: pass / blocked
Blockers:
Rollback:
```

## Owner checks

1. Install one map locally.
2. Restart from txAdmin.
3. Walk every entrance, stair, roof edge, and garage door.
4. Test with a normal player account.
5. Run `resmon` near the asset.
6. Commit only after the QA note passes.
