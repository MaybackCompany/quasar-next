## Plan before you code

Every script starts with five lines: player action, trigger, server decision, feedback, and out-of-scope.

The full cheatsheet explains the Miro flow, folder shape, server-first order, restart loop, and testing checklist.

[Open script planning](/resources/script-planning)

## First SQL ladder

1. Save one note.
2. Save one favorite location.
3. Save one transaction log.

Only add SQL when data must survive a restart.
