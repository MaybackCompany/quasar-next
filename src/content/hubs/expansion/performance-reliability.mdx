## Outcome

You will replace a tight loop with a throttled loop.

<Callout variant="tip" title="Why it matters">
  Fast loops freeze clients. Shared money changes need concurrency checks.
</Callout>

## Vocabulary

| Term | Description |
| --- | --- |
| Throttle | A delay that limits how often code runs. Example: Wait(1000) runs once per second. |
| Tight loop | A loop with no meaningful wait. Example: while true do Wait(0) for heavy work. |
| Race condition | Two actions changing the same value together. Example: two deposits at once. |

## Minimal example

```lua
local MODULE = 'reliable_loop'

CreateThread(function()
    while true do
        Wait(1000)
        print(('[QUIET][%s][tick] one safe check'):format(MODULE))
    end
end)
```
