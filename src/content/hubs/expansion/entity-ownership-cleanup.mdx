## Outcome

You will track spawned entities and clean them on resource stop.

<Callout variant="tip" title="Why it matters">
  Untracked entities leak. Leaks become server lag and visual clutter.
</Callout>

## Vocabulary

| Term | Description |
| --- | --- |
| Mission entity | An entity your script intends to manage. Example: SetEntityAsMissionEntity marks it. |
| Registry | A table that tracks created handles. Example: spawnedProps[id] = entity. |
| Cleanup path | Code that deletes entities later. Example: onResourceStop deletes props. |

## Minimal example

```lua
local MODULE = 'entity_owner'
local spawnedProps = {}

AddEventHandler('onResourceStop', function(resourceName)
    if resourceName ~= GetCurrentResourceName() then return end
    for id, entity in pairs(spawnedProps) do
        if DoesEntityExist(entity) then DeleteEntity(entity) end
        print(('[QUIET][%s][cleanup] id=%s'):format(MODULE, id))
    end
end)
```
