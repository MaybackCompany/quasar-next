## Minimal example

```lua
local MODULE = 'vehicle_persist'

RegisterNetEvent('quasar:vehicle:store', function(payload)
    local src = source
    if type(payload) ~= 'table' or type(payload.plate) ~= 'string' then
        print(('[QUIET][%s][reject] source=%s bad plate'):format(MODULE, src))
        return
    end
    MySQL.insert('INSERT INTO qu_vehicles (plate, owner) VALUES (?, ?)', { payload.plate, src })
    print(('[QUIET][%s][store] source=%s plate=%s'):format(MODULE, src, payload.plate))
end)
```

## Production checklist

1. Confirm the code is on the correct side.

4. Add one cleanup or rollback path.
