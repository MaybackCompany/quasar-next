Install the FiveM client

Before you can write a single line of Lua you need a working FiveM. Ten minutes here removes 90% of the "nothing works" feeling later. Just a download, an install, and one Cfx.re sign-in.

You'll do
Download the FiveM client, point it at your GTA V install, update game data, and sign in with Cfx.re.
Time
~10 minutes (plus download time)
You need
A legal copy of GTA V on Steam, Epic, or Rockstar Games Launcher (already updated). A working internet connection.
You'll learn
What the FiveM client is → where it lives on disk → what Cfx.re is → what "authorize" actually does

Before you start

Don't skip this. If any of these are missing, the install will silently fail and you'll be hunting for a fake bug.

Don't know where GTA V lives on disk?

Default install paths:

  • Steam: C:\Program Files (x86)\Steam\steamapps\common\Grand Theft Auto V\
  • Epic: C:\Program Files\Epic Games\GTAV\
  • Rockstar Launcher: C:\Program Files\Rockstar Games\Grand Theft Auto V\

The file you point FiveM at is GTA5.exe, not PlayGTAV.exe. Different file. Easy mix-up.

What is FiveM, in one minute

GTA V comes with one online mode, GTA Online, controlled by Rockstar. FiveM is a separate launcher that lets you connect to community-run servers instead. It uses your legal copy of GTA V to load the game world, then swaps the multiplayer code for its own.

Think of it like this: GTA V is the movie set. GTA Online is one specific show that gets filmed there. FiveM is a different production crew filming their own shows on the same set, with their own scripts, their own actors, their own rules. Same set. Different shows.

FiveM note

Your GTA V copy never changes. FiveM sits next to it as a separate program (FiveM.exe). When you launch FiveM it boots GTA V in a special mode that loads server scripts instead of Rockstar's online code. You can still play GTA Online normally any other time.

Step 1

Download the FiveM client

Outcome: FiveM.exe sitting in your Downloads folder.

  1. Open fivem.net in your browser.
  2. Click the big Download Client button.
  3. Read the Creator Platform License Agreement. Two minutes, zero filler.
  4. If you agree, click Sure, I Agree!. The FiveM.exe download starts.
Watch out

Only download FiveM from fivem.net. Other sites that claim to host "FiveM cracked" or "FiveM free" are either malware or banned mirrors. Cfx.re will lock the account that signs in from a forked client.

Step 2

Run the installer and point it at GTA V

Outcome: FiveM knows where your GTA V install lives.

  1. Open your Downloads folder. Double-click FiveM.exe.
  2. Windows may show a SmartScreen warning. Click More info → Run anyway.
  3. When prompted, navigate to your GTA V install folder and select GTA5.exe.
  4. Click Open to confirm the path.
Picked the wrong file?

If you accidentally clicked PlayGTAV.exe or any launcher shortcut, FiveM will fail with "could not find game files". Quit FiveM, re-run FiveM.exe, and pick GTA5.exe directly. The two files sit in the same folder, side by side.

Step 3

Let FiveM update game data

Outcome: FiveM has built its local cache of GTA V assets.

A message will pop up saying "local game data needs to be updated". Click Yes. FiveM will pre-process the game data so it can load servers fast. This is a one-time job.

Expect a few minutes on a fast SSD, longer on a hard drive. The progress bar moves visibly. If it freezes for more than five minutes, see the troubleshooting block below.

Why this exists

FiveM caches a stripped, signed copy of the GTA V data so it can boot into a server in seconds without re-reading every .rpf archive. You'll never see this step again on this machine.

Recap 1

Did the data update finish?

You're at a good checkpoint if FiveM's main window has opened with a server browser, even if it looks empty.

It got stuck on "updating game data"

Three usual causes:

  • GTA V is not fully patched. Open it from Steam/Epic/Rockstar once, let it patch, then quit and re-run FiveM.
  • Antivirus is scanning every file. Add an exception for the FiveM folder (usually %LocalAppData%\FiveM).
  • Disk is full or failing. Free 5 GB and try again.
FiveM closed itself with no error

Open %LocalAppData%\FiveM\FiveM.app\logs\ and read the latest CitizenFX.log. The last few lines name the cause. You'll learn to read these in Step 06.

Step 4

Sign in with Cfx.re

Outcome: a "Welcome, <your username>" screen inside FiveM.

  1. FiveM shows the Terms of Service. Read it, click I Accept.
  2. A new window appears: "Sign in with Cfx.re". Don't skip this. Without a Cfx.re account, you cannot run a server later.
  3. Click Sign in with Cfx.re. Your browser opens to the Cfx.re forum.
  4. If you don't have an account yet, that's fine, you'll create one in Step 02. For now you can come back here once you have it. If you do have one, log in as normal.
  5. Click Authorize to grant FiveM access to your Cfx.re identity.
  6. Switch back to FiveM. You'll see Welcome, <your username>. Click Continue.
What does "Authorize" actually mean?

You're telling Cfx.re: "this copy of FiveM on this machine speaks for me". The token only proves your identity to servers you join, it does not give FiveM access to your forum password, your email, or your billing info. You can revoke any time at forum.cfx.re/my/preferences/account.

Recap 2

FiveM main menu open?

If you see the FiveM server list with your username in the top right, you're done with this step. You don't need to join a public server right now, you'll connect to your own in Step 06.

I'm stuck on "sign in with Cfx.re"

The browser window opens but nothing happens? Disable any browser extensions that block third-party cookies (Brave Shields, uBlock strict mode, Cfx.re needs a session cookie). Refresh, try again.

Common mistakes & what they look like

Error: Could not find a Grand Theft Auto V installation

You pointed FiveM at the wrong folder, or at PlayGTAV.exe instead of GTA5.exe. Right-click FiveM.exe → Properties → Compatibility isn't the fix here. Just re-run FiveM.exe with GTA V closed and pick the right file.

Error: This product is not currently activated for online play

Your GTA V copy is not signed in to Steam/Epic/Rockstar, or it's a pirated copy. FiveM only works with legitimate, owned copies of GTA V. Sign in to your store, launch GTA V once, then return to FiveM.

FiveM crashes the second it opens

Outdated GPU drivers are the #1 cause. Update NVIDIA / AMD / Intel drivers from the vendor site (not Windows Update). The #2 cause is third-party game overlays (Discord, RivaTuner, Razer Synapse, AMD Adrenalin overlay). Disable overlays for FiveM, then retry.

"Welcome" screen never appears after Authorize

Browser blocked the redirect. Re-open FiveM, click Sign in with Cfx.re again, and complete authorize in the same window FiveM opened (don't copy-paste the URL into a different browser).

Cheat sheet: keep this open

FiveM.exe
The launcher. Lives in %LocalAppData%\FiveM\FiveM.exe after install.
GTA5.exe
The GTA V game binary. The file you point FiveM at on first run.
Cfx.re
FiveM's identity platform. One account = forum, server keys, license.
%LocalAppData%\FiveM\
Where FiveM stores its cache, logs, and settings on your machine.
CitizenFX.log
The crash / error log. First place to look when something doesn't work.

What you learned

  • FiveM is a separate launcher that uses your legal GTA V to connect to community servers
  • The first-run install asks for GTA5.exe, not PlayGTAV.exe
  • Cfx.re is the identity layer; one account covers the forum, server keys, and the launcher itself
  • Logs live in %LocalAppData%\FiveM\FiveM.app\logs\, that's where you go when something fails

Five-minute exercise

Open FiveM. From the main menu, click Play. Search for any public server (e.g. "Grand Theft Hamlet" or "FiveM Cops") and click Connect. You don't need to play, just confirm you can join.

Expected outcome

You see the FiveM loading screen, then either spawn into a public server or get a clear error like "you are not whitelisted". Either result proves your install + sign-in is fully working. Disconnect with F8 → quit.
