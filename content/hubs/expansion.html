Expansion modules for production FiveM

After the 28-lesson core, these modules pick up where the curriculum ends: thread budgets, native families, framework patterns, SQL discipline, persistence, and rollback.

Audience
Devs who finished the core curriculum.
Time per module
~30 min each.
Prereq
Lessons 0 through 27 complete.
Format
Strict template, runnable, audit-ready.
Use this when

Your scripts work in a sandbox but break under load, scale, or staff turnover. Each module fixes one production failure pattern.

Before you start

These modules assume you can: ship a working resource, write parameterized SQL, read a native page, and run txAdmin restarts without panic. If any of those is uncertain, finish the core curriculum first.
