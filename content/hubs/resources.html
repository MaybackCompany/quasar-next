Lua patterns the curriculum skims

Four short references for patterns that appear in real scripts but are not given a full lesson. Read each card once, copy the snippet into a sandbox resource, and confirm the expected output before using them in production code.

Apcall

Protected calls with pcall

Wrap any call that can throw (oxmysql, JSON decode, third-party exports). pcall returns ok, result so a single bad row never kills the whole event.

Lua manual: pcall

Bformat

Readable strings with string.format

Stop concatenating with .. for logs. string.format gives you typed placeholders, padding, and decimal control. The lesson logger pattern [QUIET][module][action] assumes you use it.

Lua manual: string.format

Cjson

JSON encode and decode

FiveM ships a global json table. Use it for statebag payloads, oxmysql JSON columns, and NUI messages. Always pcall decode; never trust untrusted strings.

FiveM Lua runtime reference

Di18n

Localization with locale tables

Hard-coded English breaks the moment a server runs in Spanish or French. Keep one locales/.lua per language and look up keys at call time. Falls back to English when a key is missing.
