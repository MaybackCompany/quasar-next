Module 12 - Performance

Outcome

You will replace a tight loop with a throttled loop.

Why it matters

Fast loops freeze clients. Shared money changes need concurrency checks.

Vocabulary

Throttle
A delay that limits how often code runs.
Example: Wait(1000) runs once per second.
Tight loop
A loop with no meaningful wait.
Example: while true do Wait(0) for heavy work.
Race condition
Two actions changing the same value together.
Example: two deposits at once.

Production checklist

  2. Validate every payload before acting.
  3. Log with [QUIET][module][action].
  4. Add one cleanup or rollback path.
  5. Copy the expected output into your test note.

Things to try

  • Change Wait to 5000.
  • Add a cooldown table.
  • Write one race condition risk.

Common failure modes

  • attempt to call a nil value means a native, export, or helper is unavailable on this side.
  • No such export GetCoreObject in resource qb-core means the dependency is missing or ensured late.
  • attempt to index a nil value means a table, player object, or entity handle is missing.
  • SQL error: Table 'qu_reliable_loop' doesn't exist means the migration did not run.
