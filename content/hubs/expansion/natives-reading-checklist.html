Module 03A - Natives

Outcome

You will read native docs without guessing the side, handle, or expected return.

Why it matters

Wrong-side native calls waste time. Bad handle checks create silent failures.

Vocabulary

Native
A FiveM or GTA function.
Example: DoesEntityExist(entity) checks a handle.
Handle
A number that points at an entity.
Example: vehicle is a handle.
Ownership
Who can safely control a networked entity.
Example: server logic must not assume ownership.

Setup

Resource
qu_native_check
Side
client
File path
resources/qu_native_check/client.lua
Ensure order
ensure qu_native_check

Minimal example

Expected output: [QUIET][natives][ok] appears when you sit in a vehicle.

Production checklist

  1. Confirm the code is on the correct side.
  2. Validate every payload before acting.
  3. Log with [QUIET][module][action].
  4. Add one cleanup or rollback path.
  5. Copy the expected output into your test note.

Things to try

  • Run it outside a vehicle.
  • Run it inside a vehicle.
  • Change the native and read its return type first.

Common failure modes

  • attempt to call a nil value means a native, export, or helper is unavailable on this side.
  • No such export GetCoreObject in resource qb-core means the dependency is missing or ensured late.
  • attempt to index a nil value means a table, player object, or entity handle is missing.
  • SQL error: Table 'qu_native_check' doesn't exist means the migration did not run.
