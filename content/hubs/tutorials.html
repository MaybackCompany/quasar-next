Server licensing — what you actually need

Every public FiveM server needs a license key

A free license key from the Cfx.re Keymaster is required before your server can appear in the browser list. Without one, players can still connect by IP, but your server won't be discoverable.

Free tier: Supports up to 48 player slots depending on server population. Enough for most communities just starting out.

Paid tiers: Unlock higher player caps (up to 2048 slots), priority server browser placement, and additional features. Available through Cfx.re Patreon.

The key configuration is covered in our server.cfg lesson.
