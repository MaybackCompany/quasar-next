Student projects

Business Management System for RedM

A complete boss menu, door lock, and admin panel built for a RedM roleplay server. Handles employee management, access control, and business finances. Built by a student after completing the scripting track.

Skills used: Lua scripting, MySQL, NUI, server events, framework integration

Cooperative Lumberjack Job for RedM

An advanced forestry job system with progression mechanics. Players work together to cut trees, process lumber, and deliver orders. Built on the Vorp framework with full multiplayer support.

Skills used: Entity spawning, progress bars, ox_target, event networking

Custom FiveM Server Launch

Full server built from scratch by a student — custom scripts, vehicle pack, MLO interiors, job system, and economy. Launched with a trailer and active player base within the first month.

Skills used: Server administration, resource management, monetization, community building

Custom Billboard Creator Tool

A tool that lets server owners create and place custom billboards anywhere on the map. Built by a student who wanted an easier way to add in-game advertising. Now used by multiple communities.

Skills used: NUI, entity placement, ox_lib, state management

Returning to finish what they started

Multiple students have come back for advanced programs after completing the beginner track. They started with zero Lua knowledge and returned months later to build production-ready systems. The progression works.

Skills used: Full stack FiveM development, database design, performance optimization
