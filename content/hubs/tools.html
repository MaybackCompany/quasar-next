Install these first — they unlock everything else

These 4 extensions are the foundation. Every FiveM dev uses them.

Must-have (install immediately)

  • Cfx Lua IntelliSense — FiveM natives, globals, autocomplete, and linting. The #1 extension.
  • Lua Language Server (Sumneko) — Go-to-definition, diagnostics, hover info for Lua. Works with Cfx Lua.
  • Prettier — Auto-format HTML, JSON, Markdown, and NUI files on save.
  • Material Icon Theme — Better file icons. Lua files get Lua icons. Makes navigation 2x faster.

Framework snippets

  • FiveM Lua QBCore Snippets — Auto-generate common QBCore functions. Saves typing boilerplate.
  • ESX Legacy Snippets — Code snippets for ESX-based scripts.

AI assistants

  • GitHub Copilot — AI code suggestions in real-time. Writes boilerplate, completes patterns, catches errors.
  • GitHub Copilot Chat — Chat with AI inside VS Code. "Explain this function", "Fix this bug", "Write a FiveM event listener".

Quality of life

  • Live Server — Local dev server with live reload. Essential for NUI/HTML development.
  • Auto Rename Tag — Rename opening HTML tag, closing tag renames automatically.
  • Auto Save — Saves your files automatically. Never lose work to a crash.
  • Lua Helper — Additional Lua autocomplete, syntax highlighting, and linting.
  • Code Spell Checker — Catches typos in code and comments. "recieve" → "receive".
  • Polacode — Screenshot your code beautifully. Perfect for Discord help or documentation.
