The foundation — install immediately

Every FiveM developer at Quasar uses these four tools on day one.

  • Visual Studio Code — The editor we use for all Lua, JavaScript, HTML, and configuration files. Lightweight, fast, and free.
  • FiveM Lua IntelliSense — Autocomplete for FiveM natives, globals, and events. Type GetP and see GetPlayerPed instantly. Catches typos before you run the script.
  • Lua Language Server — Go-to-definition, hover info, and diagnostics for Lua. Click any function to jump to where it's defined. Red squiggles tell you what's wrong before you test.
  • Prettier — Auto-formats your code on save. Consistent indentation, spacing, and line breaks across every file. No more arguments about style.

Framework helpers — add when you need them

Once you understand the basics, these speed up writing QBCore and ESX code.

  • QBCore Lua Snippets — Type qb and get a complete QBCore callback. Generates the boilerplate so you focus on the logic.
  • ESX Legacy Snippets — Same idea for ESX servers. Type esx and get the full pattern with placeholders.

AI assistance — your coding co-pilot

AI tools that help you write, debug, and understand code faster. Our instructors use these daily.

  • GitHub Copilot — Suggests code as you type. Write a comment like "create a blip at player position" and it generates the FiveM native calls. Learns from your codebase.
  • GitHub Copilot Chat — Ask questions in natural language. "Why is this event not firing?" or "Convert this to use ox_lib callbacks." Gets answers without leaving your editor.

Quality of life

Tools that make the daily workflow smoother. Install them as you notice the need.

  • Live Server — Opens a local web server with live reload. Essential when building NUI interfaces with HTML and CSS. Save your file, see the change instantly.
  • Auto Rename Tag — Edit an opening HTML tag, the closing tag updates automatically. Saves time when restructuring NUI layouts.
  • Code Spell Checker — Catches typos in comments and string literals. "recieve" becomes "receive" before you embarrass yourself in a commit message.
  • Material Icon Theme — Better file icons in the sidebar. Lua files get Lua icons. Config files get gear icons. Faster navigation at a glance.
