Event gateway

Client asks. Server checks every field. Server answers with a result event. The shape every safe net interaction reuses.

Purpose
Validated client to server to client round trip.
Dependencies
ox_lib (optional). FiveM server runtime.
Files
1 server-side script.
First log
[QUIET][gateway][accept] on a valid payload.
Lesson
Lesson 13: events: the four verbs
Security warning

The client can send any payload. Reject when source is 0. Validate every field type and bound. Apply a per-source cooldown before doing real work.

Files

Drop these into a new resource folder. Add the resource to server.cfg after ox_lib.

  • server.lua, net event handler with payload validation.

Install

  1. Create a folder resources/[local]/qu_gateway/.
  2. Copy server.lua into it.
  3. Add a minimal fxmanifest.lua with server_script 'server.lua'.
  4. Add ensure qu_gateway to server.cfg after ensure ox_lib.
  5. Restart the resource and trigger from a client with TriggerServerEvent('quasar:gateway:claim', { amount = 5 }).
  6. Match the log line [QUIET][gateway][accept] source=… amount=5.
