Map asset QA

Use this before adding maps, MLOs, interiors, ymap files, or large asset packs to production.

Purpose
Catch map conflicts before launch.
Dependencies
Local FiveM server, txAdmin, F8, and resmon.
Warning
Never install many maps at once.
Expected output
A pass/fail QA note with blockers and rollback step.

Copyable QA note

Owner checks

  1. Install one map locally.
  2. Restart from txAdmin.
  3. Walk every entrance, stair, roof edge, and garage door.
  4. Test with a normal player account.
  5. Run resmon near the asset.
  6. Commit only after the QA note passes.
