What should I build first?

Start with a script small enough to finish today. Do not begin with housing, gangs, phones, or a full economy.

Best first win
/coords, /myhp, or /hello.
First real feature
/heal with a server-side job check.
First world script
One prop with ox_target and cleanup.

Beginner ladder

Build in this order

  1. One command: /hello prints one log.
  2. One native: /coords prints your position.
  3. One server check: /heal asks the server before healing.
  4. One menu: open a tiny teleport menu.
  5. One prop: spawn, target, and delete one object.
  6. One SQL save: save one note and load it after restart.
  7. One vehicle: spawn one car, assign a plate, and clean it up.

Plan before you code

Every script starts with five lines: player action, trigger, server decision, feedback, and out-of-scope.
