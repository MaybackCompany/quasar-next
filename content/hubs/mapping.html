The tool that changed everything: Sollumz 2.8

Sollumz now imports and exports RAGE files (.yft, .ydr) directly from Blender. No conversion tools. No middle steps. Work in Blender, export straight to your FiveM resource. This is the plugin every mapper on our team uses.

Blender — your 3D workspace

Blender is free, open-source, and handles everything from modeling to texturing. Combined with Sollumz, it's the only 3D tool you need for FiveM asset creation.

Essential shortcuts our instructors use

These keystrokes become muscle memory after your first week. Practice them until you don't think about them.

  • G — Move an object
  • R — Rotate
  • S — Scale
  • Tab — Switch between Object and Edit mode
  • Shift + D — Duplicate
  • E — Extrude (pull new geometry from a face)
  • F — Fill (create a face from selected edges or vertices)
  • Ctrl + Z — Undo

MLO creation — build interiors that work

Creating an MLO (Map Location Object) means building an interior space that players can enter. This is how you add custom houses, businesses, police stations, and hideouts to your server.

MLO workflow shortcuts

  • N — Toggle the properties side panel
  • Shift + A — Add a new object (mesh, light, camera)
  • Ctrl + J — Join multiple objects into one
  • P — Separate selection into a new object
  • Alt + Z — X-Ray mode (see through walls while modeling)
  • Shift + Space — Maximize the viewport

Supporting tools

  • CodeWalker — Interactive 3D map viewer for GTA V. Explore the game world, find coordinates, and preview how your assets will look in context. Essential for placing MLOs at the right location.
  • Prop Database — Searchable catalog of every GTA V prop. Find any object by name or hash. Know exactly which table, chair, or decoration you want before opening Blender.
  • OpenIV — Browse and extract original GTA V game files. Useful for studying how Rockstar built the original interiors and understanding vanilla asset structure.
